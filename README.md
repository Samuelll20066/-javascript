# Fundamentos do Javascript Clássico 

## INTEGRAÇÕES

### Integrar Javascript de forma interna 

~~~ html
 <!DOCTYPE html>
    <html lang="pt-BR">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Javascript</title>
        </head>
        <body>
            <script>
                console.log("Hello world!")
            </script> 
        </body>
    </html>
~~~
